- `extract_package.py`:
본 `Transform_package` 모듈은 영화진흥위원회 오픈 API로부터 수집된 박스오피스 데이터를 변환하는 기능을 제공합니다. 이 모듈은 데이터 정제, 변환, 및 필요한 형식으로의 구조화를 포함합니다.
